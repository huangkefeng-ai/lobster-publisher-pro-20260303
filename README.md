# Lobster Publisher Pro

面向公众号创作的 AI 驱动排版工具（开源版）。

目标：做一个比常见排版器更强、更稳定、可持续迭代的公众号排版工作台。

---

## 当前进度

本项目已进入稳定迭代期，核心排版与发布能力已基本闭环：

- **魔法粘贴 (Magic Paste)**：深度兼容 Notion、飞书、Word、Google Docs。粘贴时自动将富文本/HTML 转换为标准 GFM Markdown，完美保留表格、列表、图片与行内样式。
- **多图网格 (Image Grid)**：智能识别连续图片并自动转化为美观的网格布局（支持 2-3 列自适应），无需手动调整 HTML 标签。
- **36+ 套精选主题**：内置 36 套基于 Design Tokens 的主题（如 Mac 简约、Claude 燕麦、微信原生、深林访谈等），支持一键无缝切换预览。
- **一键复制到公众号**：内置 CSS Inliner 样式自动内联管线，支持 HTML 与纯文本双重剪贴板写入，确保在微信公众号后台 1:1 还原样式。
- **响应式多端预览**：支持 PC、平板、手机三种设备尺寸实时模拟，精准把控移动端阅读体验。
- **导出能力**：支持标准 HTML 片段导出与 PDF 文档导出。
- **Markdown 增强**：支持代码块语法高亮 (Prism.js)、GFM 规范表格、任务列表 (Task Lists)。
- **全栈测试保障**：
    - **Unit Tests**: 核心解析器 (`parser`)、渲染引擎 (`renderer`)、样式管线 (`inlineStyles`) 均实现 90%+ 覆盖率。
    - **E2E Tests**: 基于 Playwright 覆盖编辑器输入、主题切换、魔法粘贴到剪贴板写入的完整链路。

---

## 已知限制

- **极窄屏适配**：在极窄的手机预览或容器下，多图网格会自动退化为单列排列以保证图片清晰度。
- **Notion 深度嵌套**：部分极其复杂的 Notion 多层嵌套 Block 在转换为 Markdown 时可能存在缩进微调需求。

---

## 快速验收清单

1. **测试魔法粘贴**：从 Notion 或飞书文档复制一段包含表格和图片的文章，直接粘贴到编辑器，应看到自动转换的 Markdown 且右侧预览正常。
2. **切换主题**：在工具栏切换 3 个以上不同风格的主题，观察预览区文字颜色、背景与引用样式的即时变化。
3. **响应式切换**：点击预览区上方的设备图标（PC/Mobile），确认文章在不同宽度下的排版表现。
4. **验证微信复制**：点击“复制到公众号”，打开微信公众平台素材编辑器进行粘贴，检查布局与图片是否完整。
5. **运行自动化验收**：
   ```bash
   npm run test:e2e
   ```

## 目录结构

- `src/core`：Markdown 解析 (`parser`)、渲染 (`renderer`) 与基础常量
- `src/editor`：编辑器 UI、状态管理与魔法粘贴监听
- `src/theme`：36 套主题注册表与 CSS 变量系统
- `src/wechat`：微信专用样式内联、剪贴板处理与内容清洗
- `src/preview`：响应式预览容器
- `src/pipeline`：核心处理管线（Markdown → HTML → Inline Styles）
